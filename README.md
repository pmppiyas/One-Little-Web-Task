# 🏠 Airbnb Clone - Frontend Project - One-Little-Web-Task

A modern, high-performance, and fully responsive **Airbnb Clone** built with **Next.js**, **Tailwind CSS**, and **TypeScript**. This project focuses on delivering a seamless user experience, mimicking the core features of the global property rental platform.

## 🚀 Live Demo

**[Live Deployment Link](https://one-little-web-pi.vercel.app)**

---

## ✨ Key Features

- **Responsive Layout:** Optimized for mobile, tablet, and desktop views.
- **Dynamic Property Listing:** Browse properties with categories and clean layouts.
- **Modern UI/UX:** Built with Tailwind CSS and Framer Motion for smooth interactions.
- **Advanced Filtering:** Category-based property filtering for better discovery.
- **Type Safety:** Developed with TypeScript to ensure robust code and fewer bugs.
- **Reusable Components:** Scalable architecture using shared UI components (Headers, Cards, Modals).

---

## 🛠️ Tools & Packages Used

### Core Technologies:

- **[Next.js](https://nextjs.org/):** React framework for production-grade web applications.
- **[TypeScript](https://www.typescriptlang.org/):** For static typing and better developer experience.
- **[Tailwind CSS](https://tailwindcss.com/):** For rapid and responsive UI development.

### Libraries & UI Components:

- **[Lucide React](https://lucide.dev/):** For consistent and beautiful icons.
- **[Shadcn UI](https://ui.shadcn.com/):** Accessible and customizable UI primitives.
- **[Zod](https://zod.dev/):** For schema validation and type-safe data handling.
- **[Framer Motion](https://www.framer.com/motion/):** For handling animations.
